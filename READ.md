# Beach Companion

A simple beach-focused web app that helps you quickly understand current and upcoming beach conditions.

## Features

- Current weather (temperature, wind)
- Tide chart and tide schedule
- Ocean water temperature
- Beach Notes (smart insights):
  - Wind shifts (onshore/offshore)
  - Rain timing and severity
  - (more coming)

## Why this app

Most weather apps show raw data.  
Beach Companion highlights what actually matters when deciding:
- Should I go now?
- Should I stay?
- Will conditions change soon?

## Supported Beaches

- Sandy Hook, NJ
- Belmar, NJ

(More coming)

## Tech

- Pure HTML / CSS / JavaScript
- NOAA + NWS public APIs
- No backend required
